# @tramio/ui

React Native UI surfaces. Subscribes to `Tour_Engine` state and emits
`UserCommand` events back into the engine. Owns route selection, tour playback
(captions, playback-speed control, high-accuracy indicator), deviation prompt,
storage management, attribution, and the Sponsored badge / disclosure pre-roll.

Module boundary set up in task 1.3. Implementation tracked under tasks 11.1–11.7.
