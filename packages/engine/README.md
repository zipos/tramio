# @tramio/engine

Pure Tour_Engine reducer. No I/O, no native bridges, no filesystem. Inputs are
`EngineEvent`s; outputs are `EngineCommand`s. This is the package property
tests P1–P12 (and P18 via capability parameterization) target.
